Política de Privacidade

Esta secção explica que dados recolhemos através do website, por que razão os recolhemos, quais os serviços terceiros envolvidos e como os tratamos.

Quem somos

A Lumen and Pixel é um estúdio de produção de eventos ao vivo baseado em Portugal. Este website é utilizado para apresentar os nossos serviços, recolher pedidos de orçamento e permitir que os visitantes subscrevam a nossa newsletter.

O que recolhemos

  • Submissões do formulário de contacto, incluindo nome, empresa ou organização, endereço de email, serviço selecionado e mensagem do projeto.
  • Subscrições de newsletter submetidas através do formulário incorporado do MailerLite.
  • Informação técnica básica necessária para a entrega do website através do GitHub Pages e das ferramentas externas listadas abaixo.

Por que razão utilizamos os dados

  • Para responder a pedidos de projeto e outras mensagens enviadas através do website.
  • Para enviar atualizações da newsletter a utilizadores que subscreveram explicitamente.
  • Para entregar recursos ou redirecionar visitantes para páginas de compra externas quando relevante.

Bases legais

  • Os pedidos de projeto são tratados com base no interesse legítimo, para que possamos responder aos pedidos recebidos.
  • As subscrições de newsletter são tratadas com base no consentimento.
  • Quando as compras são realizadas através do Gumroad, os dados de pagamento e checkout são geridos pelo Gumroad ao abrigo dos seus próprios termos e práticas de privacidade.

Serviços terceiros utilizados neste website

  • Web3Forms — utilizado para receber e encaminhar as submissões do formulário de contacto.
  • MailerLite — utilizado para subscrição de newsletter e automação de email marketing.
  • Gumroad — utilizado para páginas de produtos externos e transações.
  • GitHub Pages — utilizado para alojamento do website.

Quanto tempo guardamos os dados

Guardamos os dados de pedidos e newsletter apenas pelo tempo razoavelmente necessário para comunicação, acompanhamento operacional ou fins legais e administrativos.

Os seus direitos

Pode solicitar o acesso, correção ou eliminação dos seus dados pessoais, ou opor-se a certos tipos de tratamento, conforme aplicável ao abrigo da legislação relevante. Os emails da newsletter incluem também uma opção de cancelamento de subscrição.

Como nos contactar

Para pedidos relacionados com privacidade, utilize o formulário de contacto na página Sobre.
